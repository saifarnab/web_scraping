
Instructions:
    1. create & activate virtual env
    2. Install the dependencies (requests, bs4, lxml)
    3. Add your scrapper Api key at <API_KEY>
    4. Run -> python3 realtor.py
        a. Enter your zip code (i.e: 33312)
        b. Enter the number of bedrooms (input must be integer & greater then or equal 0)
        c. Enter the number of bathrooms (input must be integer & greater then or equal 0)
        d. Enter minimum price (input must be integer & greater then or equal 0)
        e. Enter maximum price (input must be integer, greater then or equal minimum price)
        f. Press 1 for `Buy`, Press 2 for `Rent` (input must be either 1 or 2)
        g. How many property you want extract (input must be integer & greater then or equal 1)

    5. Program will create 'rent_data.csv' or 'buy_data.csv' based on the input params
